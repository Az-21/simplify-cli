## > Simplify
**Before**
```
(火) [GroupName] Generic.Isekai_ジェネリック異世界_(_2020_)_-_S01E07.[1080p].[HEVC x265 10bit][Multi-Subs].(Weekly).mkv
```
**After**
```
Generic Isekai - S01E07.mkv
```
